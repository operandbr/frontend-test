# vue-crud-firebase project
- CRUD de usuários utilizando:
1.  VueJS
2.  Firebase
3.  Bootstrap

## Passos para executar o projeto

### Clonar o repositório
```
git clone url-do-repositorio

npm install
```

### Compilar projeto no seu ambiente local
```
npm run serve
```

### Compilar projeto (produção)
```
npm run build
```

### Executar o Lint
```
npm run lint
```
